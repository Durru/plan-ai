# Code of Conduct

## Our pledge

We want Plan-AI to be a respectful, practical, and technically rigorous community. Contributors should be direct about technical tradeoffs while remaining kind to people.

## Expected behavior

- Be respectful and constructive.
- Critique code, architecture, and decisions — not people.
- Prefer evidence, reproducible examples, and clear reasoning.
- Help newcomers understand the system instead of only pointing out mistakes.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without consent.
- Repeated disruption of technical discussion.
- Introducing unsafe code paths that knowingly risk user data or credentials.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation for behavior that violates this code. Security-sensitive reports should follow `SECURITY.md`.

# Archived spike: old Phase 7 Skill Intelligence

This directory preserves the code generated for the old Phase 7 Skill Intelligence implementation.

It was archived during **Pre-Phase 7 — Audit, cleanup, and realignment** because the project roadmap changed. The active codebase must now start the new Phase 7 with the **Definitive Domain Model**, not with Skill Intelligence.

## Why archived instead of deleted

- The spike may contain useful ideas for a future Capability Registry / Skill Intelligence phase.
- It was not aligned with the new architecture sequence.
- It introduced active CLI commands, store migration `0006_skill_intelligence`, and repository code before the definitive domain and store layers were redesigned.

## Current status

- Not compiled.
- Not imported by active code.
- Not part of current migrations.
- Reference only.

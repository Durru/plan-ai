# Planning Framework

Phase 13 creates planning artifacts from stored project state, not from raw chat messages.

Inputs:
- Vision reference
- Approved requirements, constraints, and decisions
- Stored research IDs
- Stored knowledge IDs

Artifacts:
- `MasterPlan`
- `SpecificPlan`
- `ImplementationDocument`

The framework is intentionally not an orchestrator. It records plan structure that an LLM or human can use later.

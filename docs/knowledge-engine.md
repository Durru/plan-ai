# Knowledge Engine

Phase 12 stores reusable knowledge derived from research and approved project context.

- Package: `internal/knowledge`
- Store tables: `knowledge_objects`, `knowledge_links`
- Categories: `database`, `backend`, `frontend`, `security`, `testing`, `deployment`, `architecture`, `payments`, `auth`, `general`
- Registry methods: `CreateKnowledge`, `GetKnowledge`, `ListKnowledge`, `SearchKnowledge`

Rule: existing related knowledge should be searched and reused before new research is created.

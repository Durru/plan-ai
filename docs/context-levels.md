# Context Levels

Phase 22 finalizes context delivery levels. Context is derived from project state; it is not manually duplicated.

## L0 Executive

High-level project state:

- summary
- progress
- next step
- blockers

## L1 Planning

Planning context:

- vision
- requirements
- decisions
- relevant research
- relevant knowledge

## L2 Specific Plan

Specific plan context:

- selected specific plan
- related decisions
- risks
- validations

## L3 Task

Task-level context:

- current task
- expected files
- constraints
- validations

## L4 Implementation

Implementation-level context:

- complete instructions
- implementation document fragment
- required decisions
- risks
- tests
- rollback guidance

Each delivery can be logged through `context_delivery_logs` for traceability.

# Roadmap Adjustments

## Keep

- AI-first boundary: Plan-AI orchestrates, persists, retrieves, and serves context; LLMs provide intelligence.
- Sandbox-first testing and no real OpenCode or real ~/.plan-ai mutation.
- Additive migrations only.

## Adjust

- Treat external LLM/provider adapters as a separate post-MVP integration track.
- Treat MCP production transport and OpenCode advanced integration as separate hardening tracks.
- Create a schema consolidation milestone before removing compatibility views.
- Create a baseline commit before further phases to enable real audit diffs.

## Defer

- TUI.
- Advanced Skill Intelligence.
- Engram deep integration.
- Real provider/network ingestion adapters.

## Next recommended milestones

1. Baseline commit and release candidate tag.
2. Schema consolidation plan.
3. MCP production transport hardening.
4. Real model provider adapter implementation.
5. OpenCode advanced integration design.
